# <居酒屋予約サイト>

## サイト概要
居酒屋のテーブル、コースを予約するサイト

### サイトテーマ
飲食店の予約

### テーマを選んだ理由
店側は事前に予約があれば仕込みやピークとなる時間が予想しやすくなり、
客側は前もってお店の予約ができ、店に入れないという心配がなくなる、
という便利な予約サイトを作ってみたいと思い、このテーマにしました。

### ターゲットユーザ
飲食店等の店側、飲食する客側

### 主な利用シーン
- 前もって決まっている日程で飲食店の予約をしたいとき
- 日ごとにどの程度予約が入っているか確認する
